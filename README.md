# Worm Aiva WhatsApp Bot AI

Professional WhatsApp Bot with AI capabilities using Baileys and Groq.

## Features

- **AI Integration**: Groq API for intelligent responses
- **50+ Commands**: Organized in categories
- **Multi Prefix**: Support for multiple command prefixes
- **User Management**: Premium and permission system
- **Rate Limiting**: Prevent spam and abuse
- **Auto Reconnect**: Reliable connection management
- **Database**: JSON-based persistence
- **Cache**: Efficient memory management
- **Logging**: Structured logging with different levels
- **Error Handling**: Comprehensive error management
- **Auto Backup**: Regular database backups

## Requirements

- Node.js 20.0.0 or higher
- npm 10.0.0 or higher
- Termux (Android) or any Linux environment
- Groq API Key
